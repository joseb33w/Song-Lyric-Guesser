# Lyric Drop - Song Lyric Guesser

A fast-paced music trivia game where you guess the song from lyric clues. Test your music knowledge across 50 iconic tracks, climb the global leaderboard, and challenge yourself on three difficulty modes.

## How to Play

1. Create an account with a unique username
2. Pick a difficulty: Normal (15s), Hard (10s), or Blitz (5s)
3. Read the lyric clue and pick the correct song from four choices
4. Score points based on speed and accuracy
5. Build streaks for bonus points
6. Check your rank on the global leaderboard

## Features

- 50 song clue database spanning multiple genres and decades
- 3 difficulty modes with different time limits
- Points system with streak bonuses and time bonuses
- 10 rounds per game with animated countdown timer
- Global leaderboard with sorting by points, streaks, or games played
- Personal stats dashboard with accuracy tracking and recent game history
- Real-time score updates
- Responsive mobile-first design
- Secure email authentication with confirmation flow

## Scoring

- Base points: 100 per correct answer
- Time bonus: up to 150 extra points for fast answers
- Streak bonus: 25 extra points per consecutive correct answer
- Wrong answers or timeouts: 0 points

## Tech Stack

- HTML, CSS, JavaScript (vanilla, no frameworks)
- Supabase for authentication, database, and real-time updates
- Google Fonts (Syne, Space Grotesk, JetBrains Mono)
- Font Awesome icons
- GitHub Pages for hosting
